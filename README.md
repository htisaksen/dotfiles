# .files
## Mac config to setup environment
Order of installation
1. brew
2. zsh
3. vim
4. dotfiles

## Initialization
```bash
make initialize
```
### Info
Run to initialize(brew zsh vim dotfiles makes) and install

## Install brew dependencies brew
```bash
make brew
```
